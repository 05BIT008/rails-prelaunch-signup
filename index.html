Rails Prelaunch Signup App

An example Rails 3.2 app for a web startup prelaunch site.

Download this project as a .zip file Download this project as a tar.gz file

Own the Code. But Don't Write a Single Line.

It's open source. Generate an app in minutes using an application template:

$ rails new rails-prelaunch-signup -m https://raw.github.com/RailsApps/rails3-application-templates/master/rails-prelaunch-signup-template.rb -T

You get a complete "launching soon" signup site for your web startup.
